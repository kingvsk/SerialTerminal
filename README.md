SerialTerminal
==============
Java based serial port listener.


This application is used to communicate with the devices through serial port.
You can download this project then compile and run it from Netbeans IDE.

Minimum Requirement:
====================
Java 1.6 or greater
800*600 or higher screen
Linux/Windows PC with Serial port or usb to serial converter(hub)
